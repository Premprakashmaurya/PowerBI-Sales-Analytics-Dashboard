# 📊 Power BI Sales Analytics Dashboard

## Overview

This project presents an interactive Power BI dashboard designed to analyze sales performance, profitability, customer behavior, and regional trends. The dashboard enables users to explore business insights through dynamic visualizations and filters, helping support data-driven decision-making.

---

## Business Problem

Organizations generate large volumes of sales data, making it difficult to identify profitable products, high-performing regions, and customer purchasing trends. This dashboard transforms raw sales data into meaningful visual insights for better business decisions.

---

## Objectives

- Analyze overall sales performance
- Monitor profit trends
- Identify top-performing products
- Compare regional sales
- Track customer purchasing behavior
- Create an interactive dashboard for business reporting

---

## Dashboard Features

- KPI Cards
- Sales Analysis
- Profit Analysis
- Regional Performance
- Category-wise Sales
- Customer Insights
- Interactive Filters
- Dynamic Charts

---

## Dataset

The dashboard uses two datasets:

- Orders.csv
- Details.csv
